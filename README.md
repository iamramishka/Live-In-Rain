# Live In Rain

Welcome to **Live In Rain**, a pristine UI clone and offline reimagining of the internet's most popular rain simulator experience. 

Find your peace in the storm directly in your browser or deeply integrate it into your local workspace. Completely tracker-free, open-source localized, and distraction-less.

## ✨ Features
- **100% Offline Capable**: All heavy audio files (.m4a, .ogg, .mp3), canvas bundles, and web fonts are fully downloaded into the repository. You can run this without ever pinging an external remote server.
- **Dynamic Array Backgrounds**: Unlike a static image, the background dynamically rotates between 5 beautiful, high-resolution rainy window variations upon every single page refresh. 
- **Zero Distractions**: We completely eliminated all external promotional material, App Store download badges, social media widgets, and bottom-page footer links present on the original site to give you a pure, interruption-free focus environment.
- **Privacy First**: Fully stripped out Google Tag Manager and Twitter tracking tags. 
- **Modern Typography**: Updated the UI entirely to use the clean, premium sans-serif *Outfit* font.

## 🚀 Quick Start (Local Setup)

Because the project is built purely in localized raw **HTML, CSS, and vanilla JS**, there are zero intense framework build steps. 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/iamramishka/Live-In-Rain.git
   cd Live-In-Rain
   ```
2. **Start a local dev server.** 
   You can simply use Python (which is usually pre-installed):
   ```bash
   python -m http.server 3000
   ```
   *Or via Node.js/npx:*
   ```bash
   npx serve -l 3000
   ```
3. **Open your browser** and navigate to `http://localhost:3000` to start listening.
